# AI编程入门伴学营 V3 策划包

这个文件夹用于沉淀 V3 版本的产品方案、关卡题目、系统提示词和美术资产提示词。目标是把原来的静态教程升级成一个本地伴学闯关产品：有 Key 时开启 AI 点评和导师问答，无 Key 时也能用简单模式自由学习。

## 文件说明

- `01-产品策划与关卡题目设计.md`
  - 产品定位
  - 游戏化交互
  - Key 门禁
  - 关卡结构
  - 文本作业
  - 选择题 / 判断题 / 排序题 / 场景题题库
  - 100% 通过逻辑

- `02-Art-Asset-Prompts-English.md`
  - 英文美术资产生成提示词
  - 角色、背景、徽章、UI 道具、关卡节点、特效资产
  - 面向外部图像生成 AI，可直接复制使用

- `03-Music-Generation-Prompts-English.md`
  - 英文背景音乐生成提示词
  - 启动舱、训练大厅、关卡地图、答题挑战、毕业结算等场景 BGM
  - 面向外部音乐生成 AI，可直接复制使用

- `04-音效素材清单与搜索关键词.md`
  - 中文音效需求清单
  - 交互触发点、音效风格、搜索关键词、建议文件名
  - 面向素材站检索与后期整理

- `05-首版运行说明.md`
  - 当前 V3 首版功能范围
  - 打开方式、使用流程、数据存储
  - 资源映射与已验证项目

- `06-TTS语音文本-开场与中场休息.md`
  - 后续生成 TTS 用的中文文本
  - 开场、模式介绍、中场休息、毕业挑战、结业鼓励等语音段落

- `07-GLM-TTS生成说明.md`
  - GLM-TTS 生成脚本用法
  - 环境变量设置方式
  - 输出目录和默认参数

- `scripts/generate_glm_tts.py`
  - 读取 `06-TTS语音文本-开场与中场休息.md`
  - 调用 `glm-tts`
  - 输出 `wav` 到 `assets/tts`

## 后续实现建议

`AI编程入门教程V3.html`

建议从 `Trae入门教程V2.html` 复制结构后重构，不直接覆盖 V2。V3 应使用新的 localStorage key，例如：

`ai_coding_v3_*`

避免继承 V2 的答题和解锁状态。
